# Frontend - Vintage Record (Angular 17)

Aplicación Angular con enrutamiento implementado.

## Instalación

```bash
npm install
```

## Ejecutar

```bash
npm start
```

Abre `http://localhost:4200`

## Rutas Implementadas

- `/inicio` - Página de inicio
- `/login` - Formulario de login
- `/dashboard` - Panel de administración

## Características

✅ RouterModule configurado
✅ 3 componentes con rutas
✅ `<router-outlet>` en app.component
✅ Navegación con routerLink
✅ routerLinkActive para estilos activos
